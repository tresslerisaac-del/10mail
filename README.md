# ⚡ TempMail

Disposable email + password generator. Auto-refreshes every 10 minutes. Receives real emails via [mail.tm](https://mail.tm).

## Local dev

```bash
npm install
npm run dev
# open http://localhost:3000
```

## Deploy to Railway

1. Push this repo to GitHub
2. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub repo
3. Select this repo — Railway auto-detects Node.js and runs `npm start`
4. Done. Your app gets a public URL instantly.

No environment variables needed.
